# Car Rental App Prototype

This project is a **UI/UX prototype for a Car Rental App**, developed as a **group assignment** for a school course. The goal was to design an intuitive user interface and simulate realistic user interactions for a car rental service.

---

## Overview

The app demonstrates **typical user workflows**, including:

- Browsing available cars
- Booking a car
- Viewing and managing reservations
- Handling notifications and alerts

Running the executable walks through these scenarios, showcasing the **UI/UX design decisions** made during the project.

---

## Features

- Interactive forms and screens representing a car rental flow
- Scenario-based simulations of user actions
- Demonstrates real-world usability principles

---

## Installation & Running

1. Download the release from [GitHub Releases](#).  
   - If you downloaded the single `.exe`, no additional setup is needed.
   - If you downloaded a zip of the publish folder, extract it first.
2. Double-click `.exe` file to run the app.  

> Note: This is a **prototype for demonstration purposes** — it does not connect to a live backend.
